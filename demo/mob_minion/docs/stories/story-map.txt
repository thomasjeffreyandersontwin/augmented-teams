(E) Manage Mobs
    (E) Create Mob
        (S) Game Master --> Select Minion Tokens
        and (S) Game Master --> Group Minions Into Mob
        and (S) Game Master --> Assign Mob Name
    (E) Edit Mob
        (S) Game Master --> Select Mob To Edit
        and (S) Game Master --> Add Minion To Mob
        and (S) Game Master --> Remove Minion From Mob

(E) Control Mob Actions
    (E) Select Strategy
        (S) Game Master --> Select Mob
        and (S) Game Master --> Choose Strategy
    (E) Choose Target
        (S) Game Master --> Evaluate Available Targets
        and (S) Game Master --> Apply Strategy To Select Target
        and (S) Game Master --> Confirm Target Selection
    (E) Execute Mob Action
        (S) Game Master --> Select Action Type
        and (S) Game Master --> Execute Action For All Minions
        and (S) Game Master --> Respect Range Constraints

(E) Spawn Mobs
    (E) Spawn From Template
        (S) Game Master --> Select Mob Template
        and (S) Game Master --> Spawn Mob From Template
    (E) Spawn From Actors
        (S) Game Master --> Spawn Mob From Selected Actors
